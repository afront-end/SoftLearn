# SoftLearn

Образовательная платформа для самоучек-программистов, которые хотят учиться без хаоса.

## Проблема

Самоучки теряются в море источников — YouTube, статьи, ChatGPT, форумы, курсы. Непонятно с чего начать, что важно, а что нет, и в каком порядке двигаться.

## Решение

SoftLearn даёт **один чёткий структурированный путь**: курс с фиксированным порядком, без лишних ответвлений. Никакого выбора «а что учить дальше» — платформа сама ведёт от нуля до результата.

## Как это работает

Пользователь регистрируется и проходит короткий онбординг. Если уже знает программирование — сдаёт вступительный тест, который определяет уровень и автоматически разблокирует пройденный материал. Если новичок — начинает с самого начала.

Дальше — движение по жёсткой иерархии:

**Направление → Стек → Урок → Объяснение → Практика → Тест-барьер**

Каждый стек (например, HTML&CSS, JS, TypeScript, React) открывается только после того, как сдан тест предыдущего. Перейти вперёд без сдачи теста нельзя.

Внутри каждого урока три шага:
1. **Объяснение** — материал с AI-ассистентом, которому можно задавать вопросы прямо по теме урока
2. **Практика** — задачи для закрепления (тесты, открытые вопросы, задачи с кодом)
3. **Тест-барьер** — нельзя двигаться дальше, пока не набрал минимальный порог

## Технологии

| Слой | Стек |
|------|------|
| Frontend | Next.js 14, TypeScript, Tailwind CSS, shadcn/ui |
| Backend | FastAPI, Python, PostgreSQL, Redis |
| AI | Google Gemini 2.0 Flash (чат, vision), Ollama / qwen2.5-coder (проверка кода) |
| Контент | MDX-файлы, Contentlayer, KaTeX, CodeMirror 6 |

## Структура репозитория

```
softlearn/
├── frontend/   — Next.js приложение
└── backend/    — FastAPI сервер
```

## Запуск

```bash
# Backend
cd backend
source venv/bin/activate
uvicorn main:app --reload --port 8000

# Frontend
cd frontend
npm run dev

# AI для проверки кода
ollama run qwen2.5-coder
```

Переменные окружения: `backend/.env` и `frontend/.env.local` (см. CLAUDE.md).
